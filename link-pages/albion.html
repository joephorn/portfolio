albion tool

↳ context

for my graduation project at heyheydehaas, i created a dynamic and interactive tool that generates visuals based on the albion logo and visual style, by changing parameters. albion is a swimming team that wants to be dynamic and sporty.

↳ my role

i designed and developed the tool, graphics, stickers, merch designs, and a motion visual.

↳ result

a web-based tool that heyheydehaas can use to create dynamic and sporty (motion)visuals for albion.
